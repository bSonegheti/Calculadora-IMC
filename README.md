<h1>Calculadora IMC</h1>
<h2>Objetivo da calculadora</h2>
<p>Ela tem um objetivo simples de calcular o seu peso, com a fórmula de cálculo sendo bem simples.</p>
<p>IMC = ((altura . altura) / 2 )</p>
<br>
<p>Tabela de Interpretação (Adultos)</p>
<ul>
  <li>Abaixo do peso: IMC \(<18,5\)</li>
  <li>Peso Normal: IMC \(18,5-24,9\)</li>
  <li>Sobrepeso: IMC \(25-29,9\)</li>
  <li>Obesidade (Grau I, II, III)</li>
</ul>
