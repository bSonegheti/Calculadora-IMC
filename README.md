<h1>Calculadora IMC</h1>
<h2>Objetivo da calculadora</h2>
<p>Esta aplicação tem como objetivo calcular o Índice de Massa Corporal (IMC) com base no peso e altura informados pelo usuário.</p>
<p>IMC = peso / (altura x altura)</p>
<br>
<p>Tabela de Interpretação (Adultos)</p>
<ul>
  <li>Abaixo do peso: IMC < 18,5 </li>
  <li>Peso Normal: IMC entre 18,5 e 24,9 </li>
  <li>Sobrepeso: IMC entre 25 e 29,9 </li>
  <li>Obesidade: Grau I, II e III </li>
</ul>
